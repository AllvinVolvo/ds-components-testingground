Selectable Chip

A toggle button component that uses the aria-pressed attribute to manage selection state. Commonly used for filtering options, tag selection, or representing multiple choice options in a sidebar or filter bar.

Padding Requirement: The chip-selectable class requires explicit padding utilities. Add px-16 (1rem horizontal) and py-8 (0.5rem vertical) for proper spacing.

Variants

Single Selection (Checkbox Style)

Multiple chips can be selected independently using aria-pressed attribute.

Radio Selection (Single Choice)

Use role="radiogroup" on container and aria-checked to restrict to one selection.

Disabled State

Disable chips to prevent interaction while keeping them visible.

Filter Use Case

Common pattern for filtering content across multiple categories.
